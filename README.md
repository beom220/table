# Let Make Table


### 기능
- 회원가입
- 게시글
- 댓글 

> [Client React](https://github.com/beom220/table/tree/main/client)
- Recoil을 이용한 상태관리
- MarkDown 형식으로 게시글 작성 & 프리뷰
- 이전글, 다음글 페이지네이션
- 유저권한에 따른 기능구현
- 정규식을 이용한 회원가입 인증


> [Server Express](https://github.com/beom220/table/tree/main/server)
- PassportJs를 이용한 인증방식 [Session]
- bcrypt를 이용한 유저비밀번호 암호화
- DB : Mysql
